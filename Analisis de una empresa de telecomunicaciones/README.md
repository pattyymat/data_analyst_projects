# 📱 Análisis de Clientes y Patrones de Uso

## Descripción del Proyecto

Este proyecto analiza los datos de clientes de una empresa de telecomunicaciones ficticia llamada ConnectaTel con el objetivo de comprender los patrones de uso de sus servicios, identificar segmentos de clientes y generar insights que apoyen la toma de decisiones comerciales.

## 🎯 Objetivo

Identificar diferencias en el comportamiento de los clientes según su nivel de uso y grupo etario, detectando oportunidades para mejorar la segmentación, personalización de ofertas y estrategias de retención.

## 📂 Datasets Utilizados

- users.csv
- plans.csv
- usage.csv

## 🔍 Etapas del Análisis

1. Carga y exploración inicial.
2. Limpieza y preparación de datos.
3. Integración de datasets mediante user_id.
4. Ingeniería de características.
5. Segmentación de clientes.
6. Análisis exploratorio (EDA).
7. Detección de valores atípicos.
8. Generación de insights.

## 🛠️ Tecnologías Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## ▶️ Cómo Ejecutar el Proyecto

### Google Colab
1. Descargar el notebook y los datasets.
2. Abrir Google Colab.
3. Subir el notebook y los archivos CSV.
4. Ejecutar todas las celdas.

## 🔄 Guía de Reproducción
Para reproducir los resultados:

1. Descargar los datasets originales.
2. Colocar todos los archivos CSV en la misma carpeta que el notebook.
3. Ejecutar el notebook desde la primera celda hasta la última.
4. Verificar que las rutas de los archivos coincidan con la estructura del proyecto.
5. Revisar las visualizaciones y métricas generadas para validar los resultados obtenidos.
